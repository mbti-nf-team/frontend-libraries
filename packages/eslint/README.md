## 🚀 Quick Overview

Install `@nft-team/eslint-config` dependency

```bash
npm install --save-dev @nft-team/eslint-config
```

Add `@nft-team/eslint-config` to `extends` in your `.eslintrc`

```js
module.exports = {
  // ...
  extends: ['@nft-team/eslint-config'],
  rules: {
    // ...
  },
};
```
